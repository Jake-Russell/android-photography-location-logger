Photography Location Logger User Guide

Here you can find some quick start guides on how to use the app

Navigating the App

When the app is first opened, the app's home screen is shown. This page provides easy navigation to either the Search, Add Review, Saved Locations, or User Guide screens by pressing the corresponding button.

The app's navigation menu can be accessed via the navigation icon in the top left corner. Pressing this will open the navigation drawer.

Alternatively, the screen can be swiped from left to right to access the navigation drawer.

Opening the Map for the First Time

When opening the map for the first time, you will be asked for location permissions both whilst using the app, and always. Both persmissions are required to use the full functionality of the app.

Location while using the app is required both to center the map on your location, and to add a review. Location in the background is required so that notifications can be sent out when there is a Photography Location nearby.

Searching for a Location

A location can be searched for by navigating to the app's Search screen.

This screen shows an interactable map, centered on your current location. The map can be moved by swiping in any direction, and zoomed either by pinching and zooming, or using the zoom controls in the bottom right corner.

The map can be recentered on your location by pressing the GPS icon in the top right corner. Next to this is a Map Type button. Pressing this will change the map type displayed, i.e., Hybird or Satellite.

The search bar at the top can be used to search for a location. Upon clicking on the search bar, the keyboard will appear so that the location can be entered. The Search / Enter button on the keyboard can then be pressed to move the map to this location

Adding a Review

A review of a location can be added by going to the app's Add Review screen.

Upon first opening this page, the app will check to see if there are any existing reviewd locations near you, as this could be the location that you wish to review as well. If there are any within a 500m radius of your current location, then you will be presented with a popup, asking if it is the location you wish to review. If there are multiple potential locations, then these popup suggestions will appear in order of distance from you, as to try and suggest the correct location as quickly as possible.

Pressing 'Yes' will automatically fill in the location name, so you don't need to do so. Pressing 'No' will then close the popup, so you can add your review. Alternatively, if there are other potential locations nearby, then these will be suggested too.

Only if the location you are trying to review does not already exist will you be able to provide the Location Name.

You can give the location a rating out of 5 Stars by sliding your finger along the star rating bar. The stars increment by 0.5 to allow you to give the location a precise rating.

A review can be written in the 'Review' field. The text area will automatically grow if you write more than one line, so your review can be as long and in depth as you wish!

Adding an Image to your Review

Adding an image to your review is a really quick and easy process. Simply choose between uploading an image, or taking one with your device's camera by clicking the corresponding button.

If you choose to upload an image, you will be taken to a content picker, where you will be able to choose an image from a range of different locations on your device.

If you choose to take an image, you will be taken to the camera, where you will be able to take an image. Once you have taken the image, you can accept it by pressing the tick in the bottom right corner, or retake it by pressing the cross in the bottom left corner. Note that if this is your first time using the app, or you have previously declined camera permissions, you will need to allow this permission in order to use the camera.

Once you have either selected an image from the content picker, or taken one from the camera, you will be presented with an 'Uploading' dialog. It is important that you do not click away from this dialog, as the image may not be uploaded successfully.

After the image has finished uploading, you will see a small thumbnail of it in the app. Since only one photo upload is allowed per review, you will also see that the 'Upload Image', and 'Take Image' buttons have been removed and replaced with a 'Remove Image' button. This can be pressed to delete the image, and then allow you to choose a new one.

Underneath the image, you will find a 'Submit Review' button so that you can submit the review.

Viewing all Reviews

You can view all reviews of a location by navigating to the search screen. Each reviewed location is shown on the map as a Marker Pin. Clicking on a marker pin will show a brief summary of the location, including it's name, average rating, and a button to see all reviews.

Clicking on the button will take you to a new page, where you can see all reviews of the location.

If there are any reviews of the location that have images, these will be displayed in a sliding carousel at the top. In this case, there is only one review with an image. An image can be clicked on to enlarge it. The enlarged image will be shown in a popup window. Clicking the cross in the top right corner of this popup window will close it.

An image can also be long clicked to save it to your gallery. Note that if this is your first time using the app, or you have previously declined external read / write permissions, then you will need to allow this permission in order to save the image. Once permissions have been granted, you will see another popup, asking you to confirm that you wish to save the image. Pressing 'Yes' will save the image to your gallery and close the popup. Pressing 'No', will not save the image, and close the popup.

Sharing a Location

If you've found a location that you wish to share, then this can easily be done from the All Reviews page.

In the top right corner, you will see a Share icon. Clicking this will bring up a sub-menu in which you can choose where you wish to share the location to.

In this example, the location has been shared to notes. You can see that some information about the location has been provided, including it's name, county and average rating. A Google Maps link to the location is also included, so that you can have quick access to view the precise location on Google Maps.

Seeing more Images of a Location

Sometimes, a location may not have any images associated with it's reviews. Despite this, the app does still allow you to see images of the location by performing a Google search for the location's name. This can be done by simply pressing the Image Search button in the bottom right corner.

You will be taken to Google, where you can see the image search results.

Saving a Location

If you have found a location that you really like, and wish to visit, then you can save the location in the app. This is done by pressing the Save icon in the bottom right corner.

Once saved, the icon will become filled in to indicate that the location has been saved. Pressing this again will unsave the location, and unfill the icon.

Viewing your Saved Locations

You can view your saved locations by navigating to the app's Saved Locations screen.

This will show all of your saved locations, so you know where you want to visit. Each location you have saved is shown on this screen, with it's name, average rating, and a map showing you where it is. Pressing on the location's name will take you back to the All Reviews page for that location, so you can re-read the locatoin's reviews.

Pressing on the map will allow you to open the location in Google Maps, either to get directions, or view the surrounding area.
